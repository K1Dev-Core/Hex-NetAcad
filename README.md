# Hex NetAcad - วิธีติดตั้ง
ตัวช่วยตอบ Quiz

## Chrome 
1. แตกไฟล์หรือวางโฟลเดอร์ ไว้ในเครื่อง
2. เปิดหน้า `chrome://extensions/`
3. เปิด `Developer mode`
4. กด `Load unpacked`
5. เลือกโฟลเดอร์ `dist`
6. ติดตั้งเสร็จแล้ว

## วิธีใช้งาน

1. เปิดหน้าแบบทดสอบใน NetAcad
2. จะมีปุ่มลอยมุมขวาล่าง
3. กด `เริ่ม` เพื่อให้ส่วนเสริมทำงานอัตโนมัติ
4. ถ้าต้องการหยุด ให้กด `หยุด`

## ถ้าไม่ขึ้นหรือไม่ทำงาน

1. กด `Reload` ส่วนเสริมจากหน้า Extensions
2. รีเฟรชหน้า NetAcad 1 ครั้ง
3. ลองใหม่อีกครั้ง
